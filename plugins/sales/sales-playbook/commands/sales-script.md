# Sales Script Generator

Create natural-sounding scripts for cold calls, demos, and follow-ups.

## Usage

```
/sales-script [specify: cold call, discovery call, demo, or follow-up] [product and context]
```

## Output Format

```markdown
# [Script Type] Script: [Product]

## Context
**Scenario**: [When to use this script]
**Duration**: [Expected length]
**Goal**: [Desired outcome]

---

## Opening

### Intro (First 10 seconds)
```
"Hi [Name], this is [Your Name] from [Company].
[Pattern interrupt or reason for call].
Do you have a quick moment?"
```

### If they say yes:
```
"Great, I'll be brief. [Transition to value]"
```

### If they say no:
```
"I understand. When would be a better time for a 2-minute call?"
```

---

## Value Statement

### The Hook
```
"The reason I'm calling is [value statement tied to their role/industry].

We've helped companies like [relevant example] [specific result].

I'm curious if this resonates with you?"
```

---

## Discovery Questions

### Opening Questions
```
1. "Can you tell me about how you currently [relevant process]?"
2. "What's working well? What's frustrating?"
```

### Pain Questions
```
3. "When [problem occurs], what happens?"
4. "How does that impact [relevant metric]?"
```

### Impact Questions
```
5. "If you could solve this, what would that mean for [team/company]?"
6. "Have you tried to address this before?"
```

---

## Transition to Next Steps

### If qualified:
```
"Based on what you've shared, I think we can help.
Would it make sense to schedule 30 minutes to show you
how [Company] helped [similar company] [achieve result]?"
```

### If not qualified:
```
"It sounds like [reason] might not make this a fit right now.
Would it be helpful if I sent some resources and checked back in [timeframe]?"
```

---

## Objection Responses

### "Send me an email"
```
"I'd be happy to. So I send you the right information,
can I ask one quick question first? [Ask qualifying question]"
```

### "We're all set"
```
"That's great to hear. Out of curiosity, how are you currently handling [specific challenge]?"
```

### "What's this about?"
```
"Fair question. In 30 seconds: [elevator pitch].
Does that sound like something worth exploring?"
```

---

## Closing

### Book the meeting:
```
"I have availability [Day] at [Time] or [Day] at [Time].
Which works better for you?"
```

### Send calendar invite:
```
"Perfect. I'll send over a calendar invite now.
Is [email] the best address?"
```

## Voicemail Script
```
"Hi [Name], this is [Your Name] from [Company].

[One sentence value prop relevant to them].

I'd love to share how we helped [similar company] [result].

My number is [number]. Again, that's [number].

I'll also send you a quick email. Looking forward to connecting."
```

## Tone Guidelines
- Conversational, not scripted
- Curious, not pushy
- Confident, not arrogant
- Helpful, not salesy
```
